User Guide

Add A New Assignment

On the Assignments page, tap the "+" symbol in the top right corner.

Enter the title and due date.

Click save

Delete An Existing Assignment

On the Assignments page, tap on the assignment you want to delete.

Select "Delete" on the prompt.

Edit An Assignment

Unfortunately, there's currently no edit function on assignments. We aim to implement this as soon as possible, but for now simply delete an assignment and start again by adding a new one.
